Getting Rid of Stuff

If you are cleaning out your closet, or need to get rid of some things, here are some useful places to check out if you need to buy or sell things.

  • You can place free classified ads for all sorts of things on Craigs List (Boston).
  • If you want to give things away you can post a message on the Greater-Boston Reuse list.
  • The Internet Consumer Recycling Guide has some starting points if you have potentially useful stuff that you want to get rid of.
  • If you have an old cell phone that you would like to donate look into The Wireless Foundation.
