Here are some resources that I have found useful for designing and building web pages. Some of these links are tools, some are about design, and some are resources for understanding more about how the web works.

Validation Tools

  • Bobby a utility to check your web site for accessability.
  • W3C Validation Service
  • W3C Link Checker Service
  • DoctorHTML web site checker
  • WebReference.com. Reference material for web development.

Web Site Design Resources

  • UseIt.com has lots of great advice for web site design. The site itself is a great example of simple design. The Alert Box columns are an excellent source of web design advice.
  • Web Pages That Suck. Learn from bad examples. Also, Worst of the Web.

Tools for Your Web Site

  • htDig search engine software.

Other Reference

  • CGI Environment Variables.

Information About the Web

  • Listing of Standard Assigned Port Numbers for various internet services.
  • Internet Requests for Comment or RFCs, the written definitions of the protocols and policies of the Internet.
  • The W3C page with RFCs on protocols relating to the web.
  • Cookie Central. Information about cookies and privacy.
  • An Atlas of Cyberspace has a number of map-like representations of the web that can help you visualize how hosts on the web are connected.
  • Netcraft has statistics on which what server software is used on the net.
  • The Internet Archive has a historical record of various web sites and "and other cultural artifacts in digital form"
  • The Internet Domain Survey tries to count the number of sites on the internet.
  • The Hobbes Internet Timeline lists highlights in the growth of the internet.
  • The Internet Society is an excellent resource to learn about the internet, including history and standards.

About Sites on the Net

These sites are places to learn about what is new (and old) on the web

  • The Internet Scout Project. News on resources on the internet.
  • CAIC Hoxes Page
