This site is about using software configuration management (SCM), especially version control, in a pragmatic manner as part of an agile development approach. SCM is a key part of the software development toolkit and should be considered in the context of the architecture and the team dynamics.

We believe that communication and coordination is as essential to building effective development teams as technology.
