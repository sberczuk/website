Software Configuration Management Patterns: Effective Teamwork, Practical Integration by Steve Berczuk with Brad Appleton.

This site is about agile SCM a pragmatic approach to using software configuration management (SCM), especially version control, as part of an agile development environment. SCM is a key part of the software development toolkit and should be considered in the context of the architecture and the team dynamics.

We believe that communication and coordination is as essential as technology to building effective development teams.
